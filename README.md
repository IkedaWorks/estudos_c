# **Estudos de Linguagem C**

Bem-vindo! Este repositório foi criado com o objetivo de centralizar estudos, algoritmos e projetos desenvolvidos em **Linguagem C**. O foco aqui é o aprendizado de baixo nível, lógica de programação e o entendimento de como o código interage diretamente com o hardware.

---

## 🧬 O que é a Linguagem C?

Criada por Dennis Ritchie em 1972 nos laboratórios Bell, a Linguagem C é uma das linguagens mais influentes da história da computação. Ela combina a eficiência do código de máquina (baixo nível) com a estruturação de linguagens de alto nível.

### Por que estudar C hoje?
* **Controle Absoluto:** Você gerencia diretamente a memória (ponteiros, alocação dinâmica).
* **Base de Sistemas:** É a base de sistemas operacionais (Kernel do Linux, Windows), compiladores e sistemas embarcados.
* **Entendimento Real:** Escrever em C te força a entender o que o processador está fazendo por baixo dos panos.

## 🚀 Como este repositório está organizado

* Todos os projetos são modulares ou estruturados para compilação via **GCC**.
* Arquivos temporários de IDEs (como as pastas `obj/` e `bin/` do Code::Blocks) são ignorados automaticamente via `.gitignore`.
